# Cắt ảnh sticker tự động

Web app tĩnh chạy trên GitHub Pages để tách nhiều sticker trong một ảnh lớn thành các file PNG riêng.

## Tính năng

- Chọn ảnh hoặc kéo thả ảnh vào trình duyệt.
- Tự nhận diện các sticker trên nền trắng hoặc nền sáng.
- Xem trước vùng cắt có đánh số.
- Tải từng PNG hoặc tải tất cả thành file zip.
- Có tùy chỉnh viền cắt, độ nhạy và nền trong suốt.

## Xuất bản GitHub Pages

Repo này dùng các file tĩnh ở thư mục gốc:

- `index.html`
- `styles.css`
- `app.js`

Trong GitHub, bật Pages với nguồn `Deploy from a branch`, branch `main`, folder `/root`.
